🎨 즉시 적용 가능한 Quick Wins
오늘 바로 적용하기
1. 컬러 팔레트 통일
jsx// 기존 코드에서 색상을 일관되게 교체
bg-blue-500 → bg-primary-500
bg-gray-100 → bg-neutral-100
text-gray-600 → text-neutral-600
2. 타이포그래피 업그레이드
jsx// 메인 헤드라인
<h1 className="font-display font-extrabold text-6xl text-gradient-primary">

// 부제목
<h2 className="font-display font-bold text-3xl text-secondary-900">

// 본문
<p className="text-responsive-lg text-neutral-700 leading-relaxed">
3. 버튼 스타일 개선
jsx// Primary 버튼
<button className="btn-primary hover:scale-105 transition-all duration-300">
  시작하기
</button>

// Secondary 버튼
<button className="btn-secondary hover:scale-105 transition-all duration-300">
  더 알아보기
</button>
4. 카드 컴포넌트 업그레이드
jsx<div className="bg-white rounded-2xl shadow-card hover:shadow-hover p-6 transition-all duration-300 group">
  {/* 카드 내용 */}
</div>

🔧 기술적 구현 가이드
폰트 로딩 최적화
css/* src/index.css */
@import url('https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.8/dist/web/static/pretendard.css');

/* Font display optimization */
@font-face {
  font-family: 'Poppins';
  font-display: swap;
  src: url('./assets/fonts/Poppins-ExtraBold.woff2') format('woff2');
  font-weight: 800;
}
CSS 커스텀 프로퍼티 설정
css:root {
  /* Hero 섹션용 */
  --bg-hero: #0C0C0C;
  --text-main: #FFFFFF;
  --text-accent: #7D8E82;
  --btn-primary: #7D8E82;
  --btn-primary-hover: #60796F;
  
  /* Our Story 섹션용 */
  --bg-base: #FFFFFF;
  --text-head: #26312C;
  --text-accent: #60796F;
  --line-vert: #60796F;
  --card-grad: radial-gradient(120% 120% at 120% 120%, #F9FAF9 0%, #FFFFFF 60%);
}
애니메이션 최적화
jsx// prefers-reduced-motion 고려
const prefersReducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches;

const animationConfig = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0 },
  transition: { 
    duration: prefersReducedMotion ? 0 : 0.6,
    ease: "easeOut"
  }
};
